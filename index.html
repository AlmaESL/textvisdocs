Navigate the documentation:

     - Navigate to any function or plugin in the menu to the right under Global
     - Source code to any function or plugin accessed via the link at the bottom of each plugin table 
     - Example calls given for each plugin in the documentation. For more extensive example usage,
       navigate to the "examples"-folder in the root of the GitHub repository
    
Set-up:

    1. Download the textVisBundle.js-file from the root of the GitHub repository
    2. Place the textVisBundle.js-file in the same directory as the project you work in
    3. Suitable IDE is Visual studio code, hosting your work locally via the live server extension
     
Rules for general usage:

     - How to import plugin modules from the textVisBundle.js-file: 
        Example: <script type='module'> 
                    import {barchart} from './textVisBundle.js'; 
                    import {searchWords} from './textVisBundle.js'; 
                    barchart([path1, path2], 'container', 20, 'en', 'alphabetical', ['hello', 'world'], true); 
                    const foundWords = await searchWords(myTokens, ['dummyWord1', 'dummyWord2'], 'en'); 
                    console.log(foundWords); 
                </script>
        
        This example assumes textVisBundle.js is placed in the same directory as the working project after it has been
        downloaded from the GitHub repository

     - Important: All plugins are modular-based, that is why all plugin imports and plugin calls are done inside a script of type 
        module in the above example

     - All visualization plugins require predefined div containers, with explicitly given dimensions, to be placed in: 
        Example: <div id='container' style='width:900px; height:600px;'></div>
