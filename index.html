Documentation of the TextVis plugin system

- Navigate to any function or plugin in the menu to the right
- Source code to any function or plugin accessed via the link at the bottom of each plugin table
- Example calls given for each plugin in the documentation, but for more extensive example usage navigate to the examples folder in the GitHub repository
- All plugins are modular-based, which means all plugins must be called from a script of module type
- All visualizations require predefined div containers, with explicit dimensions, to be placed in
